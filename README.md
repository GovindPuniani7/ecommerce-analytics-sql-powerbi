Ecommerce Business Intelligence Analysis (PostgreSQL + Power BI)

Project Overview

This project analyzes 100K+ ecommerce transactions to evaluate revenue performance, customer retention, growth sustainability, and geographic risk concentration.

The objective was to simulate a real-world Business Intelligence workflow using PostgreSQL for data processing and Power BI for executive reporting.

 Dataset

- 100K+ ecommerce records
- Structured into:
  - customers_raw
  - orders_raw
  - order_items_raw
- Imported using PostgreSQL COPY command

 Key Business Metrics

- Total Revenue: 15.4M
- Total Orders: 96K
- Average Order Value (AOV): 159.83
- Delivery Rate: 97%
- Repeat Customer Rate: 3%
- YoY Growth: 338%
- 37% revenue dependency on single state (concentration risk)

 Key Insights

- Growth heavily acquisition-driven (low repeat rate).
- Strong operational efficiency (97% delivery).
- Geographic revenue concentration risk identified.
- Rapid hypergrowth phase followed by plateau trend.
